Quick Start

Get Flex Audit running in seconds:

git clone https://github.com/amuzetnoM/flex_audit.git
cd flex_audit
pip install -r requirements.txt
python -m flex_audit.src /path/to/your/project

CLI Usage

Scan your codebase with advanced options:

# Basic scan
python -m flex_audit.src /path/to/project

# Generate SARIF for CI/CD
python -m flex_audit.src /path/to/project --format sarif --output ./reports

# Exclude directories and show progress
python -m flex_audit.src /path/to/project --exclude venv,node_modules --progress

# Fail on high severity findings
python -m flex_audit.src /path/to/project --fail-on high --skip-third-party

GitHub Actions Integration

Automate security audits in your CI/CD pipeline:

name: Security Audit
on: [push, pull_request]

jobs:
  audit:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      
      - name: Set up Python
        uses: actions/setup-python@v5
        with:
          python-version: '3.12'
      
      - name: Install dependencies
        run: pip install -r requirements.txt
      
      - name: Run Flex Audit
        run: python -m flex_audit.src . --format all --skip-third-party
      
      - name: Upload SARIF Report
        uses: github/codeql-action/upload-sarif@v2
        with:
          sarif_file: audit_output/audit_report.sarif
      
      - name: Upload Artifacts
        uses: actions/upload-artifact@v4
        with:
          name: audit-reports
          path: audit_output/

Configuration

Exclude Patterns: Specify files or directories to skip:

python -m flex_audit.src ./project --exclude "*.min.js,dist/*,coverage/*,venv"

Output Formats: Choose your preferred report format:

# All formats (HTML, Markdown, JSON, SARIF)
python -m flex_audit.src . --format all

# Individual formats
python -m flex_audit.src . --format html
python -m flex_audit.src . --format json
python -m flex_audit.src . --format sarif
